### 문제 설명

<p>꼬마 정민이는 이제 A + B 정도는 쉽게 계산할 수 있다. 이제 A + B + C를 계산할 차례이다!</p>

### 입력 

 <p>첫 번째 줄에 A, B, C (1 ≤ A, B, C ≤ 10<sup>12</sup>)이 공백을 사이에 두고 주어진다.</p>

### 출력 

 <p>A+B+C의 값을 출력한다.</p>

### 핵심 개념

<p>int와 long의 차이메모:문제 발생: 입력값이 $10^{12}$ (1조)인데 int를 사용하면 런타임 에러(InputMismatch) 또는 잘못된 값이 출력됨.</p>

### 해결
<p>자바의 int는 약 21억까지만 가능하므로, 그 이상의 숫자는 반드시 long 타입을 사용해야 함.</p>
