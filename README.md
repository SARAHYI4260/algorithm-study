# 📚 Algorithm Problem Solving Repository

백준과 프로그래머스 문제 풀이를 체계적으로 기록하고,
알고리즘 및 문제 해결 능력을 향상시키기 위해 운영하는 저장소입니다.

---

## 📌 Overview

이 레포지토리는 다음과 같은 목표를 가지고 있습니다:

* 알고리즘 문제 풀이 과정 및 결과 정리
* 다양한 유형의 문제 해결 경험 축적
* 일관된 학습 기록을 통한 문제 해결 능력 향상
* 코딩 테스트 대비 및 실전 감각 유지

---

## ✍️ Problem Solving Convention

* 문제 해결 후 풀이 코드 업로드
* 문제 번호 또는 문제 이름 기반 파일명 사용
* 가독성을 고려한 코드 작성 및 주석 포함
* 다양한 접근 방식 고민 및 개선
